# Virtual Assistant

# Features

- Speech Recognition
- Talk back to You
- Open Youtube and start playing a requested song 
- Tell the current time
- Tell a joke
- Open the pc camera
- Increase/Decrease Volume
- Increase/Decrease Brightness


## Dependencies

- python 3.6.8

- speechrecognition

- pyttsx3

- pyaudio (not sure if it's really needed)

### Youtube

- flask (dependency to the next one)

- pywhatkit

### Jokes

- pyjokes
